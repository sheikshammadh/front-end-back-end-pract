data types:
    a data type is a classification that specifies the type of data that a variable can hold. 
It determines the kind of operations that can be performed on the data and how the data is 
stored in memory.

3.boolean(true[1],false[0])       set{}                  range        
4.string("text")                  dict{}                 bites
5.complex(1[real],3j[imaginary])                         bytearray
